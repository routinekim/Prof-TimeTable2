# 📅 한세대학교 전임교원 시간표 시스템 매뉴얼

이 문서는 **전임교원 시간표 조회 시스템**의 일반 사용자용 및 관리자용 사용 방법을 안내합니다.

---

## 1. 일반 사용자 가이드 (조회 방법)

### 🔐 로그인 및 접속
1. 시스템 접속 시 보안을 위해 로그인 화면이 나타납니다.
2. 부여받은 **아이디**와 **비밀번호**를 입력하여 로그인합니다.
   - *아이디 예시: admin, hansei*

### 🔍 교수님 시간표 검색
1. **이름 검색**: 상단 검색창에 교수님 성함을 입력하세요.
   - 여러 명을 동시에 보고 싶다면 성함 사이에 **공백(스페이스바)**을 넣어 입력하세요 (예: `김교수 이교수`).
2. **위원회 필터**: '위원회 선택' 드롭다운 메뉴를 클릭하여 특정 위원회에 소속된 교수님들의 시간표만 모아볼 수 있습니다.
3. **태그 확인**: 시간표 셀 내의 이름 태그는 교수님별로 고유한 색상으로 표시됩니다.

### 📱 모바일 이용
- 화면 우측 상단의 **QR 코드**를 스캔하면 모바일에서도 동일하게 시간표를 조회할 수 있습니다.

---

## 2. 관리자 가이드 (데이터 관리)

관리자 권한으로 로그인하면 화면 상단에 **⚙️ Setting** 버튼이 활성화됩니다. 이 버튼을 클릭하여 관리 페이지(`committee.html`)로 이동할 수 있습니다.

### 👥 위원회 관리
1. **새 위원회 추가**: 상단 입력창에 위원회 이름을 적고 '위원회 추가' 버튼을 누릅니다.
2. **교수진 구성**: 
   - 각 위원회 카드의 텍스트 박스에 해당 소속 교수님들의 성함을 입력합니다.
   - 성함은 **쉼표(,)** 또는 **공백**으로 구분하여 입력하세요.
   - 입력 시 하단에 태그 형태로 실시간 미리보기가 제공됩니다.
3. **삭제**: 더 이상 필요 없는 위원회는 '삭제' 버튼으로 제거할 수 있습니다.

### 📊 시간표 데이터 업데이트 (엑셀 업로드)
1. 새로운 학기나 시간표 변경 시 **엑셀 파일(.xlsx)**을 통해 일괄 업데이트가 가능합니다.
2. **📁 시간표 엑셀 파일 업로드** 버튼을 클릭하여 파일을 선택합니다.
3. 업로드 즉시 클라우드 서버(Supabase)에 저장되며, 모든 사용자에게 실시간으로 반영됩니다.
   - *주의: 엑셀 첫 번째 열은 시간(교시), 이후 열은 월~금 요일 순서여야 합니다.*

### ☁️ 클라우드 동기화
- **데이터 저장**: 위원회 구성 변경 후 '📤 위원회 데이터를 서버에 저장' 버튼을 누르면 현재 설정이 서버에 영구 저장됩니다.
- **상태 확인**: 하단의 '상태' 표시줄을 통해 현재 데이터가 클라우드와 정상적으로 동기화되었는지 확인할 수 있습니다.
